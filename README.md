## 简介

**Doraemon-UI** /'dɔ:ra:'emɔn/，简称DoUI，中文名 哆啦A梦UI，意味着能够像哆啦A梦一样提供各种各样的UI组件及工具。

> [English README](README_EN.md)

## 开发背景

**简单总结**

1、Doraemon-UI 提供全面丰富的UI组件，能够快速让你的网站快速成型，提高生产力；

2、Doraemon-UI 内置很多常用的工具，避免重复实现，一次接入，你将会拥有强大的工具集合。
